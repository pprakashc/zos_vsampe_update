# zos_vsampe_update

Repository containing the REXX script `peupdate.rexx`.

## Overview
- Script: `peupdate.rexx` (located at repository root)
- Purpose: Source REXX script; repository stores metadata and usage notes.

## Requirements
- A REXX interpreter (e.g. Regina REXX or ooRexx) installed on your machine.

## Running
Open a shell in the repository and run with your REXX interpreter. Examples:

```
regina peupdate.rexx
```
or

```
rexx peupdate.rexx
```

If the script requires arguments or environment configuration, add them after the script name.

## Repository layout
- `peupdate.rexx` — main script (kept at repo root)
- `docs/` — project documentation
